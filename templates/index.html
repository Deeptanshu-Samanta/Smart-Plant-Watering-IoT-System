Process

How it works.

From sensor to action in milliseconds.

  1. 1
    Sense
    ESP32 reads soil moisture, temperature, and humidity and POSTs to the Flask server.
  2. 2
    Predict
    Random Forest model evaluates all sensor values and predicts whether the pump should be ON or OFF.
  3. 3
    Log
    Only when the pump state changes, the event is saved with a reason, timestamp, and duration.
  4. 4
    Visualize
    Dashboard shows live KPIs, event timeline chart, and the full state change history.
